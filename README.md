Python version 3.12.4 used from anaconda navigator. 

For engine calculations: 
1) import AeroEqs.py into a python file
2) use AeroEqs.engineCalculations()
3) for non-standard gamma and cp: engineCalculations(gammaA, gammaG, cpa, cpg)

These are equations used in the following aerospace classes: aerospace computing; gas dynamics; flight mechanics; air-breathing propulsion; rocket propulsion
